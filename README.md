# LifeCompass AI

Ready-to-upload static website for GitHub Pages.

## Upload
Upload these files to the root of your GitHub repository:

- index.html
- icon.svg
- manifest.webmanifest

GitHub Pages should redeploy automatically after committing the files.

## Important
The chat on this version is an interactive preview. It is not connected to a live AI service and does not collect or save personal information.
